¿Que es HTML?
HTML (HyperText Markup Language) es el lenguaje usado para crear páginas web. Sirve para estructurar el contenido, como textos, imágenes, enlaces y tablas, que luego el navegador muestra al usuario.
¿Que es CSS?
CSS (Cascading Style Sheets) es el lenguaje que se usa para **dar estilo y diseño** a las páginas web creadas con HTML, como colores, fuentes, márgenes y disposición de los elementos.
